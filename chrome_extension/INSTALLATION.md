# Quick Installation Guide

## Step-by-Step Instructions

### 1. Extract the Extension
- Unzip the `f1-fantasy-exporter.zip` file to a location on your computer
- Remember where you saved it!

### 2. Open Chrome Extensions
- Open Google Chrome
- Type `chrome://extensions/` in the address bar and press Enter
- OR click the three dots menu ‚Üí More Tools ‚Üí Extensions

### 3. Enable Developer Mode
- Look for the "Developer mode" toggle in the top-right corner
- Turn it ON (it should turn blue)

### 4. Load the Extension
- Click the "Load unpacked" button that appears
- Navigate to the `f1-fantasy-exporter` folder you extracted
- Select the folder and click "Select Folder" (or "Open")

### 5. Pin the Extension (Optional but Recommended)
- Click the puzzle piece icon in Chrome's toolbar
- Find "F1 Fantasy Data Exporter" in the list
- Click the pin icon next to it
- The extension icon will now appear in your toolbar

## Using the Extension

1. Go to https://fantasy.formula1.com/
2. Navigate to the team selection page where you can see drivers
3. Click the F1 Fantasy Exporter icon in your toolbar
4. Click "Export Drivers" or "Export Constructors"
5. The CSV file will download automatically!

## Tips

- **For Constructors**: Click the "Constructors" tab on the F1 Fantasy page BEFORE clicking Export Constructors
- **Price Changes**: Negative values mean the price went down, positive means it went up
- **Updates**: If you make changes to the extension files, go to chrome://extensions/ and click the refresh icon

## Need Help?

If the extension doesn't work:
1. Make sure you're on fantasy.formula1.com
2. Refresh the page
3. Make sure the driver/constructor list is visible
4. Check the Chrome console (F12) for error messages
